NameDescription
Home PageThe application presents a home page with a search box prompting the user to search for product names, type, or brand.
AutocompleteAs soon as the user has typed m letters, an autocomplete box show the n closest matches among all product categories, names, or brand, where m and n are parameters.
Initial SearchAfter the user clicks on the search submit button, Creco displays a list of related categories, with the message "What kind of product are you looking for?"
Feature DisplayAfter the user selects a category, Creco displays all the features relevant to this category, ranked in order of importance, where importance can be customized. Each feature is associated with textual explanations of some form and a text box.
Raw Product DisplayAfter the user selects a category, Creco displays all the products in this category, ranked in order of relevance, where relevance can be customized. Products names are linked to their product page on the Consumer Reports website.
Feature-sensitive Product DisplayAnytime the user selects/unselects a feature in the list, the list of products is re-ranked by weighting more heavily products that have a good rating/score for the selected features.
Feature Selection ResetIt is possible to de-select all features at once, which triggers the Raw Product Display.
